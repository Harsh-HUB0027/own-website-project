
Hotel Management Project Website (School submission)
Files included:
- index.html
- style.css
- script.js
- images/ (placeholder images)
- README.txt

How to use:
1. Extract the zip file.
2. Open index.html in your browser (double-click).
3. To publish on GitHub Pages:
   - Create a new GitHub repository.
   - Upload the extracted files (root should contain index.html).
   - In repo Settings -> Pages, select branch 'main' (or 'master') and save.
   - Your site will be available at https://yourusername.github.io/repo-name/

Notes:
- The site content is original and inspired by a layout similar to Shiksha's informational pages for NCHMCT JEE.
- External official resources (exam PDF) are linked directly from government or authoritative pages.
